# 무료 배포 가이드 — Render(백엔드) + Vercel(프론트엔드)

완료하면 이런 URL로 어디서든 접속 가능합니다:
- 프론트엔드: https://stock-analyzer-여러분.vercel.app
- 백엔드 API: https://stock-analyzer-backend.onrender.com

---

## 사전 준비 (5분)

필요한 계정 3개를 미리 만들어두세요 (모두 무료):
- GitHub: https://github.com/signup
- Render: https://dashboard.render.com/register
- Vercel: https://vercel.com/signup (GitHub 계정으로 로그인 가능)

---

## STEP 1 — API 키 발급 (5분)

1. https://platform.claude.com/dashboard 접속
2. 왼쪽 메뉴 → **API Keys** 클릭
3. **Create Key** → 이름 입력 (예: "stock-analyzer") → 생성
4. 키를 **즉시 복사**해서 메모장에 저장
   - `sk-ant-api03-...` 형태로 시작
   - 창을 닫으면 다시 볼 수 없음
5. 왼쪽 메뉴 → **Billing** → 카드 등록 + 소액 크레딧 충전 ($5 권장)

---

## STEP 2 — GitHub에 코드 올리기 (5분)

### 2-1. 폴더 구조 확인
```
stock-analyzer/
├── backend/
│   ├── main.py
│   └── requirements.txt

│   ├── index.html
│   └── src/
│       ├── main.jsx
│       └── App.jsx
└── render.yaml
```

### 2-2. Git 초기화 & GitHub 업로드

터미널(또는 Git Bash)에서:

```bash
# 프로젝트 폴더로 이동
cd stock-analyzer

# Git 초기화
git init
git add .
git commit -m "첫 번째 커밋"
```

GitHub 웹사이트에서:
1. https://github.com/new 접속
2. Repository name: `stock-analyzer`
3. **Private** 선택 (API 키 보안을 위해)
4. **Create repository** 클릭

터미널로 돌아와서:
```bash
git remote add origin https://github.com/본인계정/stock-analyzer.git
git branch -M main
git push -u origin main
```

---

## STEP 3 — Render에 백엔드 배포 (10분)

1. https://dashboard.render.com 접속
2. **New +** → **Web Service** 클릭
3. **Connect a repository** → GitHub 연결 → `stock-analyzer` 선택
4. 설정 입력:
   - **Name**: `stock-analyzer-backend`
   - **Root Directory**: `backend`
   - **Runtime**: `Python 3`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `uvicorn main:app --host 0.0.0.0 --port $PORT`
   - **Instance Type**: `Free` 선택
5. **Environment Variables** 섹션:
   - Key: `ANTHROPIC_API_KEY`
   - Value: `sk-ant-...` (STEP 1에서 복사한 키)
6. **Create Web Service** 클릭

배포에 3-5분 소요됩니다. 완료되면:
`https://stock-analyzer-backend.onrender.com` 같은 URL이 생깁니다.

> ⚠️ 무료 플랜은 15분 미사용 시 슬립 모드 진입.
> 첫 요청 시 30초 정도 대기 후 깨어납니다.

---

## STEP 4 — Vercel에 프론트엔드 배포 (5분)

1. https://vercel.com 접속 → GitHub으로 로그인
2. **Add New Project** 클릭
3. GitHub에서 `stock-analyzer` 저장소 Import
4. 설정:
   - **Root Directory**: `frontend` (중요!)
   - **Framework Preset**: Vite (자동 감지됨)
   - **Build Command**: `npm run build` (자동 입력됨)
   - **Output Directory**: `dist` (자동 입력됨)
5. **Environment Variables** 추가:
   - Name: `VITE_API_URL`
   - Value: `https://stock-analyzer-backend.onrender.com` (STEP 3 URL)
6. **Deploy** 클릭

2-3분 후 배포 완료:
`https://stock-analyzer-여러분.vercel.app` 접속 가능!

---

## STEP 5 — 테스트 (2분)

배포된 URL 접속 → 종목 코드 입력 → 분석 시작

처음 실행 시 Render 백엔드가 슬립 상태일 수 있어서
"연결 중..." 메시지가 30초 정도 나올 수 있습니다. 정상입니다.

## 문제 해결

| 증상 | 해결 |
|---|---|
| 백엔드 연결 안됨 | Render 대시보드에서 로그 확인 |
| CORS 오류 | backend/main.py의 CORS 설정 확인 |
| API 키 오류 | Render 환경변수에서 키 재입력 |
| Vercel 빌드 실패 | Root Directory가 `frontend`인지 확인 |
